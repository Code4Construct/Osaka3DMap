How to use KojiPDF

動画と画像で分かりやすく解説。インストールからPDF出力まで、ステップごとにご案内します。
Follow the steps below — from installation to exporting your merged PDF.

ステップガイド / Step-by-Step Guide
1
Step 01
アプリを起動する
Launch the Application

インストール後、デスクトップのショートカットまたはスタートメニューから KojiPDF を起動します。
After installation, launch KojiPDF from the desktop shortcut or Start menu.

スクリーンショット / Screenshot
アプリ起動画面 · App launch screen
ここをクリック / Click here — デスクトップのアイコンをダブルクリックしてアプリを起動 · Double-click the desktop icon to open the app.
2
Step 02
フォルダを選択する
Select Your Folder

「フォルダを選択」ボタンをクリックし、結合したいPDFファイルが入ったフォルダを選択します。サブフォルダも自動的に検索されます。
Click the "Select Folder" button and choose the folder containing the PDFs you want to merge. Subfolders are scanned automatically.

スクリーンショット / Screenshot
フォルダ選択ダイアログ · Folder selection dialog
「フォルダを選択」ボタン / "Select Folder" button — ウィンドウ上部のボタンをクリックしてダイアログを開く · Click the button at the top of the window to open the folder browser.
Tip
フォルダ構造がそのままPDFのしおり階層になります。整理されたフォルダ構成を用意しておきましょう。
Your folder structure becomes the bookmark hierarchy in the merged PDF — organize your folders before merging.
3
Step 03
ファイルリストを確認する
Review the File List

選択したフォルダ内のPDFファイルが一覧表示されます。結合順序やしおりの構造を確認してください。Word・Excel・PowerPointファイルは自動的にPDFに変換されます。
All PDF files in the folder are listed. Review the merge order and bookmark structure. Word, Excel, and PowerPoint files are automatically converted to PDF.

ファイルリスト
File list view
しおりプレビュー
Bookmark preview
左側：ファイルリスト / Left: File list — 結合されるファイルの順序を確認できます · Check the order files will be merged.   右側：しおり構造 / Right: Bookmark tree — 最終的なPDFのナビゲーション構造 · Preview of the final PDF navigation.
4
Step 04
出力設定をする
Configure Output Settings

出力ファイル名と保存先フォルダを設定します。必要に応じてPDFの品質やしおりの設定を変更できます。
Set the output file name and destination folder. You can also adjust PDF quality and bookmark options as needed.

スクリーンショット / Screenshot
出力設定パネル · Output settings panel
出力先を選択 / Choose destination — 「参照」ボタンで保存先フォルダを指定し、ファイル名を入力してください · Click "Browse" to set the save location and type your desired file name.
5
Step 05
PDFを生成する
Generate the PDF

「PDF生成」ボタンをクリックすると処理が開始されます。ファイル数によって処理時間が異なります。完了するとダイアログで通知されます。
Click the "Generate PDF" button to start processing. Time varies by file count. A dialog will notify you when complete.

スクリーンショット / Screenshot
処理中の画面 · Processing / progress screen
「PDF生成」ボタン / "Generate PDF" button — ウィンドウ下部の大きなボタンをクリック。プログレスバーで進捗を確認できます · Click the large button at the bottom. A progress bar shows the current status.
Note
Word・Excel・PowerPointファイルが含まれる場合、Microsoft Officeがインストールされている必要があります。変換は自動で行われます。
If Office files are included, Microsoft Office must be installed. Conversion happens automatically in the background.
6
Step 06
完成したPDFを確認する
Review the Output PDF

生成が完了したら、PDFビューアで開いてしおりの構造を確認します。フォルダ名が親しおり、ファイル名が子しおりとして表示されます。
Once complete, open the PDF in your viewer to check the bookmark structure. Folder names appear as parent bookmarks; file names as child bookmarks.

スクリーンショット / Screenshot
完成PDF・しおりパネル · Final PDF with bookmark panel
しおりパネル / Bookmark panel — PDFビューアの左側にしおりパネルを表示して、階層構造を確認してください · Open the bookmark/outline panel on the left side of your PDF viewer to verify the hierarchy.
